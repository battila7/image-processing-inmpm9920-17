# Exercise 2

## Prerequisites

  * Create an input document containing english text. 
  * Save this into an image file. 

## Noise

Create a program that adds additive noise to this image. The noise generator should have one parameter
  * percent - how much of the original pixels should be toggled.

## Filter

Create a filter (using windowing technique) which can be either mean or median.

## Histogram

Implement histogram equalizing.
